# Paper Workspace

This directory contains research-facing material for the recommendation
reasoning project.

```text
notes/          Method notes and research framing.
manuscript/     Future paper source.
figures/        Paper figures generated from code or result summaries.
tables/         Paper tables generated from verified experiment outputs.
references/     BibTeX files and local PDFs.
reviews/        Internal review notes and revision ledgers.
submission/     Venue checks and submission package notes.
```

Current method note: [`notes/method.md`](notes/method.md).

Only add citations after checking the source. Keep unverified claims in notes
until an experiment file or citation supports them.
